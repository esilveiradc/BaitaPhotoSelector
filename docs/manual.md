# Manual do Usuário

1. Abra o programa.
2. Selecione uma pasta contendo as fotos.
3. Escolha se deseja incluir RAW.
4. Clique em Analisar.
5. Revise as fotos suspeitas.
6. Use os botões para manter ou enviar para a pasta de fotos ruins.
7. Utilize Comparar Parecidas para escolher a melhor foto de uma sequência.

A análise automática serve como triagem inicial e não substitui a revisão final do fotógrafo.
